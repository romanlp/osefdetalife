# Prep: Switch to Master

## RULES

- Execute this step before any story creation work begins.

## INSTRUCTIONS

1. Switch to the `master` branch: `git checkout master`
2. Pull the latest changes: `git pull origin master`
3. Confirm the branch and latest commit to the user.
